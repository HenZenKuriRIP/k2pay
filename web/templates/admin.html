官方支付 · 支付宝商户

接入说明：将支付宝开放平台「当面付 / 预下单 alipay.trade.precreate」接入本系统。 模式选「官方接口」后，用户扫官方收款码支付，异步回调自动入账；配置不全或下单失败时自动回退个人收款码。
回调地址：https://你的域名/api/channel/notify/alipay
必须公网可访问（建议 HTTPS），用于拼官方异步回调 URL
官方模式需在开放平台开通「当面付」并配置应用公私钥

官方支付 · 微信商户

接入说明：接入微信支付商户平台 APIv3 Native 扫码（/v3/pay/transactions/native）。 模式选「官方」后生成微信官方收款码，支付成功异步回调自动入账；失败时自动回退个人收款码。
回调地址：https://你的域名/api/channel/notify/wechat
需在微信商户平台开通 Native 支付，并配置 API 证书与 APIv3 密钥
在商户平台「账户中心 → API 安全」设置，必须 32 位
